# Movie Rating System (Java)

Projeto simples em Java criado para praticar conceitos fundamentais da linguagem.

## Objetivo

Simular um pequeno sistema de filmes no terminal, utilizando:
- variáveis
- condicionais
- loops
- leitura de dados do usuário

## Funcionalidades

- Exibir informações de filmes
- Verificar condições usando if/else
- Ler dados digitados pelo usuário
- Calcular média de avaliações

## Tecnologias

- Java
- IntelliJ IDEA

## Conceitos praticados

- Tipos primitivos
- Estruturas condicionais
- Estruturas de repetição
- Classe Scanner
- Operadores aritméticos
